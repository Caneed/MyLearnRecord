## 变量声明

- 不声明类型:

```dart
var name='小明'
```

- 声明类型:

```dart
String name2='小明'
```

- 类型不明确时可以使用`dynamic`

```dart
dynamic name3 = '小明';
```

## 默认值

变量声明之后如果不赋值，它的默认值为 null，即使时是数字类型也是 null，在 dart 中所有变量都是对象。

## final 和 const

创建在使用过程中不会改变的变量，可以使用`final`或者`const`，`final`变量的值只能被设置一次，`const`在编译时就已经被固定（const是隐式final类型）
