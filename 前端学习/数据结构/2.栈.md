# 栈

[TOC]

栈是一个先进后出的数据结构

js 中没有栈这个数据类型，可以通过 array 来模拟栈的操作

```js
const arr = []
for (let i = 0; i < n; i += 1) {
  arr.push([])
  for (let j = 0; j < n; j += 1) {
    arr[i].push(j)
  }
}
```

### 十进制转换为二进制

```js
const dec2bin = num => {
  // 创建栈

  let stack = []
  let res = ""

  // 当十进制数大于0时进入循环
  while (num > 0) {
    stack.push(num % 2)
    num = num >> 1
  }
  // 出栈
  while (stack.length) {
    res += stack.pop()
  }
  return res
}
```
