### 回调地狱

如果在回调函数中再传入一个函数，层层嵌套就形成了回调地狱。

比如在发送多个Ajax请求时，第二个请求需要第一个的结果中的值作为参数，第三个请求需要第二个结果中的值作为参数。。。

层层嵌套的回调就是回调地狱

```
//有多个异步任务，要求需要同时拿到所有异步任务的结果，下边就是用回调地狱
$.get("url", (res1) => {
    conosle.log(res1)
    $.get("url+res1", (res2) => {
        conosle.log(res2)
        $.get("url+res2", (res3) => {
            conosle.log(res3)
            $.get("url+res3", (res4) => {
                conosle.log(res4)
            })
        })
    })
})
```
