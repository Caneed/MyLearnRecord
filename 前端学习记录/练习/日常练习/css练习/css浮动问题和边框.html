父元素塌陷问题

子元素和父元素同时有margin属性时，子元素不会相对父元素移动

可以为父元素指定border，或者给父元素加上内边距，又或者可以为父元素加上overflow:hidden

清除网页元素的内外边距

在浏览器中有默认的内外边距，清除它可以：
*{ padding:0; margin:0; }

圆角边框

border-radius:数值,就可以将边框变成圆角

圆角边框的盒子

将盒子变成圆形border-radius:50%;

圆角矩形：将border-radius设置为高度的一半

border-radius可以设置四个值。分别为左上，右上，左下，右下

还可以后面设置两个数值。分别为左上加右下，右上，加左下
